アプリについて

司法試験予備試験の短答式過去問を演習できるアプリです。令和元年〜令和7年の全科目に対応しています。

お問い合わせ

不具合の報告や機能のリクエストは、GitHub Issues からお願いします。

お問い合わせはこちら

よくある質問

  • 購入が反映されない場合 — アプリ内の「購入を復元」ボタンをお試しください。
  • 問題の内容に誤りがある場合 — GitHub Issues からご報告ください。

プライバシーポリシー

本アプリは個人情報を収集しません。解答履歴はすべて端末内にのみ保存され、外部サーバーへの送信は行いません。アプリ内課金の処理は Apple のシステムを通じて行われます。
