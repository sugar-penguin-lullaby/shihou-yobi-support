# shihou-yobi-support

iOSアプリ「予備試験 短答式過去問」の GitHub Pages 公開用リポジトリです。  
AdMob の `app-ads.txt` 対応を目的としています。

## 公開ファイル

- `index.html`（日本語の1ページサイト）
- `app-ads.txt`（公開ルート直下）

## app-ads.txt

```txt
google.com, pub-4136704154693953, DIRECT, f08c47fec0942fa0
```

## 想定公開URL

- https://sugar-penguin-lullaby.github.io/shihou-yobi-support/
- https://sugar-penguin-lullaby.github.io/shihou-yobi-support/app-ads.txt

## GitHub Pages 設定

1. GitHub で `shihou-yobi-support` リポジトリを作成
2. このリポジトリを push
3. GitHub の `Settings > Pages` で `Deploy from a branch` を選択
4. Branch を `main` / Folder を `/ (root)` に設定
